# Toko Buah Total

## Description

Toko Buah Total memiliki 3 macam stok buah, yaitu Apel, Jeruk dan Anggur

## Feature

Fitur dari program ini adalah melihat stok, menambah daftar buah, mengubah data buah, menghapus data buah dan belanja buah!
